Note.js
Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices. This allows nodes to be written in JSX or even C++! It also supports the use of extensions with the help of CommonJS modules that allow you to extend Node's functionality using other packages like http, https and fs modules .
Javascript
JavaScript is a programming language that allows you to do all kinds of stuff, including making websites. It's used in many popular web browsers, like Chrome and Firefox, so it's easy to learn how to use it. JavaScript is written in a programming language called JavaScript, which was created by Brendan Eich and first released in 1995. It's an interpreted language meaning it can take the instructions written in JavaScript and run them on your computer or device. JavaScript is a very flexible language that lets you create different things depending on what you've got available in the browser.
React
React is a JavaScript library that helps you build user interfaces. You can use it to create React components and render them on the screen. You can also use it to manage state, which is the data that your UI uses to keep track of what it's showing and how it should behave. The state of a React component is stored in variables, which are declared inside the component, so they're accessible to any child components.
